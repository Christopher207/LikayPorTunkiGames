¿Qué es EduCards AR?

EduCards AR es una plataforma educativa de vanguardia que fusiona el mundo físico con experiencias digitales inmersivas. Mediante el uso de tarjetas físicas especiales y tecnología de realidad aumentada, transformamos conceptos abstractos en modelos 3D interactivos que los estudiantes pueden explorar, manipular y comprender de manera profunda.

📱 Escanea

Cada tarjeta contiene un código QR único que, al ser escaneado con un dispositivo móvil, proyecta un modelo tridimensional relacionado con el tema de estudio.

🎨 Visualiza

Los estudiantes pueden rotar, acercar y explorar células eucariotas, estructuras atómicas, ecosistemas completos o cualquier concepto educativo.

🎯 Aprende

Más allá de la visualización, incluye quizzes interactivos, analíticas de progreso y un panel de administración para educadores.

Construido con Tecnología de Vanguardia

Arquitectura limpia, escalable y lista para producción

Backend: Go + Gin Framework
Database: PostgreSQL
Auth: JWT
Frontend: React 18 + TypeScript
3D: Three.js
Styling: Tailwind CSS
Mobile: React Native + Expo
AR: WebAR
DevOps: Docker + Nginx
CI/CD: GitHub Actions
Cloud: AWS S3 / MinIO
API: Google Drive

Características Principales

Una plataforma completa para crear experiencias educativas inmersivas

🎓 Panel de Administración

Gestiona tarjetas, modelos 3D y contenido educativo con una interfaz intuitiva

📊 Dashboard Analítico

Métricas en tiempo real y reportes de aprendizaje para tomar decisiones informadas

JWT seguro con roles admin/estudiante para control de acceso

Evaluaciones interactivas con puntuación y retroalimentación inmediata

☁️ Feedback Automático

Sincronización con Google Sheets para seguimiento automático del progreso

Casos de Uso

Aplicaciones prácticas en diferentes áreas educativas

🧬 Biología

Tarjeta de célula eucariota → Modelo 3D interactivo con orgánulos. Los estudiantes pueden explorar cada componente celular en 3D y comprender su función.

⚛️ Física

Tarjeta de átomo → Visualización 3D de estructura atómica. Observa los electrones, protones y neutrones en movimiento para entender la teoría atómica.

🌍 Geografía

Tarjeta de ecosistema → Exploración AR de bosque amazónico. Vive una experiencia inmersiva en el corazón de la selva tropical.
